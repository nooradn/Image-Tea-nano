## Install & Clone

1. **Install Git**: Unduh di [git-scm.com](https://git-scm.com/download/win).
2. **Buka CMD/PowerShell**.
3. **Pilih folder project**:

   ```bash
   cd /d D:\Project
   ```

   atau ke Desktop:

   ```bash
   cd %USERPROFILE%\Desktop
   ```
4. **Clone repo**:

   ```bash
   git clone https://github.com/mudrikam/Image-Tea-nano.git
   ```
5. **Masuk folder**:

   ```bash
   cd Image-Tea-nano
   ```

---

## Jalankan Aplikasi
